Control System and Data Collection of Mobile Robot through Webserver

Purpose & Future

This is a system that collects data from Mobile Robot through the MQTT protocol to transfer data from Esp32 to the server on Raspberry. From there Raspberry will read the data, process and send it to MySQL to store the data. The web server is responsible for reading data from SQL, displaying it through charts and tables for users to easily monitor.

It is also possible to control the robot's movement direction directly on the webserver in real-time.

Robots can be applied in life with tasks such as searching and collecting environmental data in places that are difficult for humans to reach.

In the next version, the robot will be equipped with cameras, temperature sensors, gas sensors,... to be able to perform more tasks.
